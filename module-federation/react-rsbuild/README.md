# React + Rsbuild Module Federation

A Module Federation setup with React and Rsbuild, demonstrating micro-frontend architecture with a provider/consumer pattern, deployed with Zephyr Cloud.

## Tech Stack

- React 19
- Rsbuild
- Module Federation
- Zephyr Cloud

## Quick Start

```bash
npx degit ZephyrCloudIO/zephyr-examples/module-federation/react-rsbuild my-app
cd my-app
pnpm install
pnpm dev
```

## Deploy

```bash
pnpm build    # Builds and deploys to Zephyr Cloud
```
