To run this example:

In `zephyr-mono`'s root directory, run each command in succession:

```
pnpm i
pnpm --filter vite-remote run build
pnpm --filter vite_rspack run build
pnpm --filter vite_webpack run build
pnpm --filter vite-host run build
```
